# 🚀 Sparx Cookie Getter (Improved)

A lightweight and refined browser extension that extracts only the required Sparx cookies for SolverX, with a clean UI and streamlined experience.

# ✨ Features
🔍 Automatically captures only the necessary cookies:
live_ssoprovider_session
spxlrn_session
🎯 No unnecessary data collection (avoids full cookie dump)
🧼 Clean, simple, and responsive UI
⚡ Fast one-click extraction
🧩 Works on Chromium-based browsers (Chrome, Edge)
# 📦 Installation
Download or clone this repository
Extract the .zip file (if downloaded)
Remove the README.md file (optional, avoids loading issues in some cases)
# 🛠️ Setup
Open your browser extensions page:
Chrome → chrome://extensions
Edge → edge://extensions
Enable Developer Mode (top right)
Click Load unpacked
Select the extracted folder
# ▶️ Usage
Go to the Sparx website and log in
Click the Extensions (🧩 puzzle icon) in the top-right corner
Select Sparx Cookie Getter
Click Get Cookie
Your required cookies will be displayed or copied (depending on implementation)
